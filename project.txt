# 🗳️ AI-Powered Election Assistant (PromptWars Challenge - Google x Hack2Skill)

## 📌 Project Overview

This project is an AI-powered assistant designed to help users understand the election process in a clear, interactive, and personalized way. It simplifies complex election procedures, timelines, and eligibility requirements into an easy-to-follow conversational experience.

The system leverages modern AI/ML techniques, including Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and agent-based orchestration to provide context-aware and dynamic responses.

---

## 🎯 Objective

Build a smart assistant that:

* Explains election processes step-by-step
* Provides personalized guidance based on user context
* Displays timelines and key election events
* Ensures accuracy using verified data sources
* Supports real-world usability and accessibility

---

## 🧠 Core Capabilities

### 1. Personalized Election Guidance

* Tailors responses based on:

  * User location (country/state)
  * First-time voter status
* Generates a structured journey:

  * Registration → Verification → Voting

### 2. Context-Aware AI Assistant

* Maintains conversation memory
* Adapts responses dynamically based on user inputs

### 3. Election Timeline Generator

* Provides:

  * Registration deadlines
  * Voting dates
  * Result announcements

### 4. Interactive Flow-Based UI

* Visual step-by-step representation of the voting process

### 5. Multi-Mode Explanation

* Standard Mode (detailed explanation)
* Simplified Mode ("Explain Like I'm 10")

### 6. Misinformation Detection (RAG-based)

* Validates responses using trusted election data sources
* Reduces hallucinations

### 7. Voice Interaction (Optional)

* Speech-to-text for user queries

### 8. Multi-language Support

* Supports multiple regional languages for accessibility

## 🏗️ System Architecture

### High-Level Architecture

Frontend (Next.js)
↓
Backend API (FastAPI)
↓
LangGraph Agent Layer
├── LLM (Gemini API)
├── Vector Database (RAG)
├── PostgreSQL (structured data)
└── Firebase (user session & memory)

---

### 🔁 AI Agent Workflow

1. User Input
2. Intent Classification

   * Timeline Query
   * Process Guidance
   * Eligibility Check
3. Context Enrichment

   * User profile
   * Location
4. Routing Layer

   * Directs query to appropriate tool
5. Tool Execution

   * Timeline Tool
   * Registration Guide Tool
   * FAQ Retriever (RAG)
6. Response Generation (LLM)

## ⚙️ Functional Workflow

Example Flow:

User Query: "How do I vote in India?"

System Actions:

1. Detects user location
2. Checks user profile (new voter or not)
3. Retrieves:

   * Registration steps
   * Deadlines
   * Required documents
4. Generates structured response
5. Optionally simplifies explanation
